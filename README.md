# 🚀 Travique - Backend API Server

This project was developed as **Project 2: Backend Development** for the DecodeLabs Full Stack Developer Internship.

The objective of this phase was to build a scalable backend server capable of handling API requests, managing travel destination data, and preparing the application for database integration.

---

## 📌 Project Overview

Travique Backend serves as the core engine of the application. It provides RESTful APIs that allow the frontend to communicate with the server and manage destination-related operations.

The backend is designed with a modular structure to ensure maintainability, scalability, and clean code organization.

## 📡 API Endpoints

| Endpoint | Method | Description |
|-----------|----------|-------------|
| /api/destinations | GET | Fetch all destinations |
| /api/destinations | POST | Add a new destination |
| /api/destinations/:id | DELETE | Delete a destination |
